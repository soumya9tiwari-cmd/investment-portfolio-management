# Investment Portfolio Management

## Overview

This project presents a comprehensive investment portfolio management framework developed using Modern Portfolio Theory (MPT) and the Treynor-Black model.

The objective was to design an investment strategy tailored to an investor's risk profile by determining the optimal capital allocation, constructing an efficient portfolio, selecting securities, and evaluating portfolio performance using risk-adjusted measures.

---

## Project Objectives

- Assess investor risk tolerance
- Develop an Investment Policy Statement (IPS)
- Determine the optimal capital allocation
- Optimize portfolio asset allocation
- Construct an actively managed equity portfolio
- Evaluate portfolio performance using multiple risk-adjusted metrics

## Methodology

The portfolio construction process follows a top-down investment approach:

1. Determine investor risk tolerance
2. Allocate capital between risky and risk-free assets
3. Optimize the allocation between equity and fixed income
4. Select securities using the Treynor-Black model
5. Evaluate portfolio performance using risk-adjusted performance measures
